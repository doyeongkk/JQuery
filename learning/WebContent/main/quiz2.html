[문제 1] 다음은 static에 대한 설명이다. 틀린 것은?

2. static 형 변수는 클래스 로딩시에 메모리가 할당되어 프로그램 종료까지 그 영역이 고정된다.
3. static 메서드 안에 선언되는 변수들은 모두 static 변수가 된다.
4. static 메서드 안에서는 this나 super를 사용할 수 없다.
